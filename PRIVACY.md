# 개인정보 처리방침 — LocalAI Bench

최종 업데이트: 2026-06-15

LocalAI Bench(이하 "앱")는 사용자의 개인정보를 존중합니다. 이 앱은 **온디바이스(기기 내)**에서 동작하며, 개인 데이터를 수집하거나 외부로 전송하지 않습니다.

## 1. 수집하는 정보
앱은 어떤 개인정보도 수집하지 않습니다.
- 회원가입·로그인 없음
- 분석(애널리틱스)·광고·추적 없음
- 사용자 식별자(디바이스 ID 등) 수집 없음

## 2. 데이터 저장
- **채팅 대화와 측정 기록**은 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.
- 앱을 삭제하면 저장된 모든 데이터가 함께 제거됩니다.

## 3. 네트워크 통신
앱은 다음의 경우에만 네트워크에 연결합니다.
- **AI 모델 다운로드**: 사용자가 모델을 받을 때 HuggingFace(huggingface.co)에 연결합니다. 이때 HuggingFace의 개인정보 처리방침이 적용됩니다(IP 주소 등 일반적인 웹 요청 정보).
- **모델 카탈로그 업데이트**: 모델 목록을 GitHub에서 내려받습니다.

사용자의 대화 내용이나 측정 데이터는 어떤 경우에도 외부로 전송되지 않습니다.

## 4. 제3자 서비스
- HuggingFace — AI 모델 다운로드
- GitHub — 모델 카탈로그 호스팅

이들은 파일을 내려받는 용도로만 사용됩니다.

## 5. 생성형 AI 고지
앱의 답변은 온디바이스 AI 모델이 생성하며, 부정확하거나 부적절할 수 있습니다. 중요한 결정에 그대로 사용하지 마세요. 부적절한 콘텐츠는 아래 이메일로 신고해 주세요.
