# 개인정보 처리방침 — LocalAI Bench

최종 업데이트: 2026-06-16

LocalAI Bench(이하 "앱")는 사용자의 개인정보를 존중합니다. 이 앱은 **온디바이스(기기 내)**에서 동작하며, 개인 데이터를 수집하거나 외부로 전송하지 않습니다.

## 1. 수집하는 정보
앱은 어떤 개인정보도 수집하지 않습니다.
- 회원가입·로그인 없음
- 분석(애널리틱스)·광고·추적 없음
- 사용자 식별자(디바이스 ID 등) 수집 없음

## 2. 기기 권한 사용
앱은 아래 권한을 **해당 기능을 사용할 때만** 요청하며, 모든 처리는 기기 안에서 이루어집니다.
- **카메라**: 비전(이미지) 모델의 실시간 인식 테스트에 사용합니다. 카메라 영상은 기기 내에서만 분석되며 저장·전송하지 않습니다.
- **마이크**: 음성 인식(STT) 모델 테스트에 사용합니다. 녹음은 기기 내에서 텍스트로 변환되며, 음성·텍스트 모두 외부로 전송되지 않습니다. 임시 녹음 파일은 인식 후 기기 내에서 처리됩니다.
- **사진 보관함**: 사용자가 선택한 이미지를 비전 모델 입력으로 사용합니다. 선택한 이미지만 기기 내에서 처리됩니다.
- **파일**: 사용자가 선택한 문서·이미지·오디오 파일을 기기 내에서 처리합니다(문서·오디오는 텍스트로 변환).

이 권한들은 모두 선택 기능이며, 거부해도 나머지 기능은 정상 동작합니다.

## 3. 데이터 저장
- **채팅 대화, 측정 기록, 음성 변환 텍스트**는 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.
- 카메라 영상과 마이크 녹음은 인식 처리에만 사용되며, 영구 보관하지 않습니다.
- 앱을 삭제하면 저장된 모든 데이터가 함께 제거됩니다.

## 4. 네트워크 통신
앱은 다음의 경우에만 네트워크에 연결합니다.
- **AI 모델 다운로드**: 사용자가 텍스트·비전·음성(Whisper) 모델을 받을 때 HuggingFace(huggingface.co)에 연결합니다. 이때 HuggingFace의 개인정보 처리방침이 적용됩니다(IP 주소 등 일반적인 웹 요청 정보).
- **모델 카탈로그 업데이트**: 모델 목록을 GitHub에서 내려받습니다.

사용자의 대화·이미지·음성·측정 데이터는 어떤 경우에도 외부로 전송되지 않습니다.

## 5. 제3자 서비스
- HuggingFace — AI 모델 파일 다운로드
- GitHub — 모델 카탈로그 호스팅

이들은 파일을 내려받는 용도로만 사용됩니다.

## 6. 생성형 AI 고지
앱의 답변·인식 결과는 온디바이스 AI 모델이 생성하며, 부정확하거나 부적절할 수 있습니다. 중요한 결정에 그대로 사용하지 마세요. 부적절한 응답은 **앱 안에서 길게 눌러 신고**할 수 있고(신고 시 즉시 숨김), 아래 이메일로도 신고할 수 있습니다.

## 7. 아동 보호
앱은 제한 없는 AI 생성 콘텐츠를 포함하므로 **만 17세 이상**을 권장합니다. 13세 미만 아동의 개인정보를 고의로 수집하지 않습니다.
